📋 Panduan Setup Google Sheets

  1. Buka Google Sheets → buat spreadsheet baru → salin ID dari URL (bagian antara /d/ dan /edit)
  2. Buka Extensions → Apps Script
  3. Hapus semua kode yang ada, lalu paste seluruh isi file Code.gs yang diunduh bersama halaman ini
  4. Di baris pertama file, isi SPREADSHEET_ID dengan ID spreadsheet Anda
  5. Klik Deploy → New Deployment → pilih type: Web App
  6. Set: Execute as: Me dan Who has access: Anyone
  7. Klik Deploy → izinkan akses → salin URL Web App
  8. Paste URL tersebut di kolom konfigurasi di atas, klik Simpan
  9. Klik Inisialisasi Data Default untuk membuat user & data awal
Login default: admin/admin123 · manager/manager123 · warehouse/warehouse123 · guest/guest123
⚠️ Untuk embed di Google Sites: Insert → Embed → By URL → masukkan URL halaman ini. Pastikan halaman ini di-host (mis. GitHub Pages, Netlify, atau Google Drive).
